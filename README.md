# CV Portfolio Website

A modern, responsive portfolio website built with Next.js 14 and Tailwind CSS. This frontend-only website features direct WhatsApp integration for freelance services.

## Features

- ✅ Fully responsive design
- ✅ WhatsApp integration for all services
- ✅ Smooth scrolling navigation
- ✅ Modern UI with Tailwind CSS
- ✅ No backend required
- ✅ Contact form with WhatsApp redirection
- ✅ Portfolio showcase
- ✅ Skills & experience sections
